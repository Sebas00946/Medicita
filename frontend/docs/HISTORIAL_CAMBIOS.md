# Historial de Cambios - Frontend MediCita Web

## Versión 1.1.0 - 23/05/2026

### Nuevas Funcionalidades
- ✅ **AdminLayout**: Sidebar profesional para Administrador con menú completo
- ✅ **DoctorLayout**: Sidebar para Médicos con tema verde personalizado
- ✅ **Página de Gestión de Médicos**: CRUD completo con modal de nuevo/editar
- ✅ **Página de Reportes**: Estadísticas y citas recientes con diseño moderno
- ✅ **Login y Registro rediseñados**: Diseño responsive, gradients, iconos de Lucide React
- ✅ **Navegación inteligente**: Layout automático según el rol del usuario

### Nuevas Rutas
- `/admin/dashboard` - Dashboard del Administrador
- `/admin/medicos` - Gestión de Médicos
- `/admin/reportes` - Reportes y Estadísticas
- `/admin/configuracion` - Configuración (en construcción)
- `/admin/citas` - Gestión de Citas (en construcción)
- `/medico/citas` - Mis Citas (Médico, en construcción)
- `/medico/disponibilidad` - Mi Disponibilidad (Médico, en construcción)
- `/medico/perfil` - Mi Perfil (Médico, en construcción)
- `/medico/reportes` - Mis Reportes (Médico, en construcción)

### Mejoras
- Uso de iconos profesionales de Lucide React en toda la app
- Diseño moderno con Tailwind CSS
- Responsive design para todas las pantallas
- Mejora en la experiencia de usuario
